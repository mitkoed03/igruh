# Ten Worlds

Game for the GMTK 2026 Game Jam with theme "Countdown".

## Idea

Platformer. There's a given number of "wrongdoings" in a level. Your job is to figure what they are and fix them as most as possible without going negative. A few worlds with different mechanics and graphics.
